# Nested-Comments
Threaded comments like on Reddit or Youtube

For Example: 
@X  <-> Reply to X
#Y  <-> IdComm Y

@0 #1 Comment
   @1 #2 Comment
      @2 #5 Comment
      @2 #8 Comment
   @1 #11 Comment

   @3 #6 Comment
   @3 #9 Comment
      @9 #10 Comment
      @9 #4 Comment
